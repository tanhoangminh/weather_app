# weather_app

## Cài đặt môi trường

- Flutter SDK: v3.13.7 trở lên
- Dart SDK: v3.1.3 trở lên
- Android Emulator: Android 12.0+ (API 31)
- Java OpenJDK v11.0
- Cocoapods v1.12.0


## Chạy chương trình
- `git clone https://github.com/tanhoangminh/weather_app.git`
- `cd weather_app && flutter run --debug`
